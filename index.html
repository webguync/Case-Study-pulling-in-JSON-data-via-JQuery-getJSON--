Sandbox - Learning JSON

Info: In this case study, I wanted to learn more about JSON and how you can pull in JSON formatted data using JQuery getJSON() method.

I decided to use the last four U.S Presidents and use te following information. Their official portrait, first and last name, Inauguration month, day and year.

Pretty easy to do once you have the hang of it. The JSON formatting can be a bit tricky. A great tool I discovered is JSONLint to check your JSON formatting.

Now that I have the basics of pulling in JSON code in HTML via JQuery, I am confident I can move into much more complex JavaScript application development.

Check out the code I used to create this case study on my GitHub account.
